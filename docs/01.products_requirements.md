# 상품 목록 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 쇼핑몰 상품 목록 페이지


### 사용하게 될 기술: React


### 문서 작성자: 박유진

## 📝 고객 요구사항 정리
 ﻿(1) 웹페이지를 기반으로 하되 모바일에서도 사용이 편리하게 요청
 (2) 상품목록 페이지에서 상품사진(신발 사진)이 잘 보이게 요청
(3) 직원들도 테스트해 볼 수 있게 링그 공유 요청
